📋 Visão Geral

Um sistema avançado e modular para identificação automática de juntas de solda, classificação de tipo de passe (raiz, enchimento, acabamento) e seleção inteligente de processo de soldagem (SMAW, GTAW, GMAW, FCAW) usando visão computacional e ROS. O projeto combina técnicas de Deep Learning, OpenCV e arquitetura robótica modular para automação industrial.

🎯 Objetivos Principais

  • Detecção Automática: Identificar juntas de solda em imagens com precisão > 95%
  • Classificação Inteligente: Determinar o tipo de passe — raiz, enchimento ou acabamento
  • Seleção de Processo: Escolher entre SMAW, GTAW, GMAW, FCAW com base em parâmetros de engenharia
  • Integração ROS: Arquitetura completamente modular com 3 nós, 19 tópicos e 5 serviços
  • Bem Documentado: Quick Start, Arquitetura e documentação API completa

💻 Uso Direto via Python

python 
from joint_detector import WeldingJointDetector, WeldingClassifier
import cv2

# Carrega imagem da junta
image = cv2.imread("junta.jpg")
detector = WeldingJointDetector()
joint = detector.detect_joint(image)

if joint:
    classifier = WeldingClassifier()
    plan = classifier.plan_welding(joint)
    print(f"Processo: {plan.process.value}")
    print(f"Passe: {plan.pass_type} | Confiança: {plan.confidence:.1%}")
